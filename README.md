# Як працює Програма

Програма запозичує легальні посилання onion з текстових файлів або з словника програми. 

# Команди 

`/a пошукові системи, /b сайти,/restart перезапуск, /exit вихід ,/info інформація, /wikionionss вікі-onion, /i2p. посилання i2p`
